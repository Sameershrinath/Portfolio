UNIV APP

HTML CSS JavaScript Firebase Auth.

Project Overview

UNIV is a web-based application designed to enhance the college experience by enabling seamless interaction among students. The platform provides secure user authentication, personalized profiles, real-time chat, and collaborative study tools, all powered by Firebase and modern web technologies.

Key Features

  • Secure Authentication: Sign up and log in using Firebase Authentication with email and password.
  • Personalized Profiles: Each user has a customizable profile with editable information.
  • Real-time Chat: Communicate instantly with other students via Firebase's real-time database.
  • Study Tools: Create and manage study groups, and collaborate on academic projects.

Tech Stack

  • HTML, CSS, JavaScript
  • Firebase Authentication & Realtime Database
